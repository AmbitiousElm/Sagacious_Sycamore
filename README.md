## Usage

Adjust live polling:

In `server.js` set `setInterval(poll, 5000);` to desired interval.

Test polling:

Send `GET` request to `/api/test`

## Requirements

- Node
- Bower
- MySQL


## Setup


Setup in the root folder:

```
npm install

```

Start Server in a new terminal tab in the root folder:

```
npm start

```

Visit:

```

localhost:3000

```
